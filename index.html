Research Background & Domain

Our research bridges autonomous robotics, deep learning, IoT sensing, and cloud computing to address a persistent real-world problem — waste management in high-traffic public spaces.

🤖
Deep Learning for Waste Detection
YOLO-based object detection models (YOLOv4, YOLOv8) demonstrate high accuracy and real-time performance for identifying waste materials under diverse environmental conditions. CNN-based approaches achieve high accuracy when trained on large datasets such as TACO and Kaggle garbage classification datasets.
📡
IoT-Enabled Waste Monitoring
Integrating IoT-enabled sensing with deep learning significantly improves monitoring and decision-making in waste management systems. Smart bins with ultrasonic sensors and cloud connectivity enable real-time fill-level monitoring and notification — but are mostly limited to stationary applications.
🛸
Autonomous Mobile Robotics
Sensor-based techniques using ultrasonic and infrared sensors provide cost-effective obstacle detection and collision avoidance. Sensor fusion with GPS and inertial data enhances localization accuracy. Existing robots (OATCR, AutoClean) lack full integration for dynamic railway environments.
🔍
Comparison of Waste Detection Methods
Existing models trade off accuracy vs. computational efficiency — large models perform better but are unsuitable for embedded systems; lightweight models reduce accuracy. No prior unified solution addresses end-to-end autonomous waste management in railway environments.

A significant research gap exists in developing a fully integrated, railway-specific autonomous waste management system that combines real-time waste detection, robotic collection, adaptive navigation, safety-aware operation, bin-level monitoring, and intelligent unloading within a single platform.

Aspect Existing Systems Research Gap Identified
Navigation & Obstacle Avoidance SLAM-based mapping, predefined routes, ultrasonic/IR sensors for collision avoidance Effective in structured indoor settings but inadequate for semi-structured, dynamic railway environments; insufficient attention to drop-off prevention
Platform Safety (Drop-off Detection) IR and ToF sensing for cliff detection in domestic/service robots Not adapted for railway platforms with higher safety risks and crowded conditions
Unloading Mechanisms Semi-automatic or human-triggered disposal; docking via QR codes Requires environmental modifications, reducing practicality in busy transport hubs
Localization for Disposal GPS-based positioning with sensor fusion (GPS + IMU + odometry) GPS degrades in semi-covered railway platforms; hybrid solutions underexplored
Bin-Level Monitoring IoT-enabled static bins with ultrasonic fill sensors and cloud alerts Mostly limited to stationary bins; rarely integrated with mobile robots

How to architect autonomous mobile robots that can strongly navigate semi-structured, safety-critical environments such as railway stations, steer clear of collisions, guard against falls from platforms, and locate disposal points exactly with GPS-based location awareness, in order to support completely automatic waste unloading?

⚠️
Safety-Critical Navigation
Railway stations present dynamic environments with fluctuating pedestrian density, unexpected obstacles, and elevated platforms providing unique safety hazards — factors not addressed in existing indoor robot navigation systems.
📍
GPS-Challenged Environments
Semi-covered railway platforms degrade GPS signal quality. Discovering and driving to a definitive disposal location, aligning correctly, and controlling an unloading system autonomously remains an insufficiently researched area.
01
Autonomous Garbage Detection and Collection
Design and implement a robust garbage detection and collection mechanism combining computer vision (YOLOv8) and sensor-based triggering. A robotic arm with appropriate degrees of freedom picks and stores identified waste items with high detection accuracy and reliable grasping.
02
Intelligent Navigation & Unloading Control
Develop a reliable navigation system using ultrasonic and infrared sensors for obstacle detection and platform-edge prevention. Implement intelligent unloading triggered by real-time bin-level monitoring — the robot navigates to disposal points via GPS only when necessary.
03
Real-Time Crowd Detection & Waste Classification
Develop ML-based crowd density estimation via head-count detection to support safe navigation decisions (slowdown, reroute, pause in congested areas). Simultaneously classify collected waste into plastic, paper, glass, and polythene at the disposal station.
04
Automated Waste Sorting & Disposal Bin Monitoring
Implement automated waste sorting at the disposal station using a robotic arm. Deploy IoT-based bin monitoring (ultrasonic sensors) with cloud transmission, real-time visualization, and automated alerts when bins approach capacity limits.

Methodology & Results

System Architecture & Performance

Four integrated subsystems — mobile robot, cloud, disposal area, and mobile application — work in seamless synergy to achieve fully autonomous waste management.

SUBSYSTEM 01
Mobile Robot Subsystem
The core active unit for waste collection. Equipped with ESP32-CAM and ultrasonic sensors for real-time waste detection. Upon detection, the robotic arm (1 kg capacity) picks and stores waste. Navigates autonomously using reactive sensor-driven logic, with platform-edge detection via IR sensors and crowd-aware safe-zone protocol.
ESP32-CAM Ultrasonic IR Sensors GPS Robotic Arm
SUBSYSTEM 02
Cloud Subsystem
Central intelligence and connectivity layer hosting the waste classification ML model, crowd detection model, and cloud database. Connects the robot, disposal area, and mobile app via MQTT and HTTP. Enables long-term trend analysis, predictive maintenance, and scalability across multiple stations.
MQTT HTTP Cloud DB ML Models
SUBSYSTEM 03
Disposal Area Subsystem
A stationary robotic arm guided by ML classification output sorts unloaded waste into four dedicated bins (Plastic, Paper, Glass, Metal). Each bin has an ultrasonic fill sensor reporting levels to the cloud in real time, enabling timely maintenance alerts and preventing overflow.
Stationary Arm 4-Class Sort IoT Sensors MQTT
SUBSYSTEM 04
Mobile Application
Human-machine interface providing real-time visibility of disposal bin fill levels. Automatic push notifications are sent when bins near capacity, prompting timely action. Closes the loop between fully autonomous operation and human monitoring oversight.
Real-Time UI Push Alerts Bin Status
